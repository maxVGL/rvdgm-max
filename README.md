# Minijuego Max

Proyecto de un juego de lucha hecho con react. Permite elegir de una serie de personajes, en un gameplay donde el RNG lo decidirá todo.

## Instalación

Antes de descargar, crear una carpeta con `npx create-react-app rvdgm-max`.

A continuación, descarga el projecto e inserta todo dentro de la carpeta realizada.

## Dependencias

Antes de iniciar la app, ejecutar estos dos comandos:
 - `npm install bootstrap`
 - `npm install react-router-dom`

Esto instalará las dos dependencias principales que se necesitan para ejecutar la app.

## Ejecución

Iniciar la app con `npm start`, y se abrirá en el navegador.
